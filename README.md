# Online Shop

A simple e-commerce website built with HTML, CSS, and JavaScript, allowing users to browse products, add them to a cart, and simulate a checkout process.

## Features

- **Product Catalog**: Display a list of products with images, descriptions, and prices.
- **Shopping Cart**: Add/remove items from the cart, view total price.
- **User Authentication**: Basic login/signup simulation (frontend only).
- **Responsive Design**: Works on desktop and mobile devices.
- **Search Functionality**: Filter products by name or category.

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
- **Styling**: Custom CSS with responsive design
- **Data Storage**: Local Storage for cart persistence (no backend)

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/EROLVX/online-shop.git
   ```

2. Navigate to the project directory:
   ```
   cd online-shop
   ```

3. Open `index.html` in your web browser to run the application locally.

No additional dependencies or server setup required, as this is a frontend-only project.

## Usage

- Browse products on the homepage.
- Click "Add to Cart" to add items.
- View your cart by clicking the cart icon.
- Proceed to checkout (simulated).
